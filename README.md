# 大量台語名姓搜揣

作者：[陳建中｜ Tân Kiàn-tiong](https://www.kiantiong.com)

做一改搜揣大量的台語名姓，名姓讀音資料源自[教育部臺灣台語常用詞辭典](https://sutian.moe.edu.tw/und-hani/piantsip/pankhuan-singbing/)，原始資料遵照[創用 CC 姓名標示-禁止改作 3.0 臺灣 授權條款](https://creativecommons.org/licenses/by-nd/3.0/tw/)授權使用。

網頁程式有倚靠 Cursor 鬥相共，程式碼的部份以 [CC0](https://creativecommons.org/publicdomain/zero/1.0/deed.zh-hant) 開源授權，無著作權。

## 離線使用說明

這个網頁會當離線使用，共原始碼[Táng-lóo](https://github.com/luke871016/taigi-names/archive/refs/heads/main.zip)落來，佇你的設備用瀏覽器拍開就會用得執行矣喔。

## 家己增加讀音

若是你有需要，修改 `embedded-data.js` 的內容，會當家己增加讀音資料。
