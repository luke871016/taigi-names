# 大量台語名姓搜揣

作者：[陳建中｜ Tân Kiàn-tiong](https://www.kiantiong.com)

做一改搜揣大量的台語名姓的網頁家私

## 離線使用說明

這个網頁會當離線使用，共原始碼[Táng-lóo](https://github.com/luke871016/taigi-names/archive/refs/heads/main.zip)落來，佇你的設備用瀏覽器拍開就會用得執行矣喔。

## 家己增加讀音

若是你有需要，修改 `embedded-data.js` 的內容，會當家己增加讀音資料。

## 網頁程式

網頁程式多數倚靠 Cursor 鬥相共，程式碼的部份並無遐專業，猶是以 [CC0](https://creativecommons.org/publicdomain/zero/1.0/deed.zh-hant) 開源授權，無著作權，隨在你提去用。

## 讀音資料

名姓讀音資料源自[教育部臺灣台語常用詞辭典](https://sutian.moe.edu.tw/und-hani/piantsip/pankhuan-singbing/)，原始資料遵照[創用 CC 姓名標示-禁止改作 3.0 臺灣 授權條款](https://creativecommons.org/licenses/by-nd/3.0/tw/)授權使用。

## 網頁字型

網頁字型是國家太空總署的訂做字型 [TASA Orbiter](https://github.com/localremotetw/TASA-Typeface-Collection)，遵照 [SIL Open Font License](https://openfontlicense.org/) 授權使用。有關 TASA 字型的介紹會當參考[建中以早寫的文章](https://kiantiong.com/jihing/tasa_typeface_collection_intro/)。
